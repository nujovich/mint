mint.
v0.3.0
July 20, 2026

Five new commands, four new export targets, and a Penpot bridge — mint's biggest release yet.

New commands

  • mint-ds apply — codemod source CSS straight to tokens
  • mint-ds lint — flag a11y and gap-decoration CSS issues
  • mint-ds score — grade your CSS health at a glance
  • mint-ds init — scaffold a config in seconds
  • mint-ds diff — compare two token files

Export targets

  • SolidJS and Qwik targets
  • UnoCSS target
  • Vanilla Extract target
  • --target dtcg deterministic export

Smarter CSS & interop

  • Interop 2026 safe-adoption rules
  • Modern, a11y-focused layout linting
  • Penpot token import plugin (DTCG)

Bug fixes

  • WSL2: retry LLM calls and force IPv4 DNS
  • Ship compat/lint modules in the npm tarball
